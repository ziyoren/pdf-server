ZIYO PDF Server

This is a simple nodejs script that uses the html-pdf library to convert an HTML file into a PDF.

API

POST /pdf Show PDF

Parameters x-www-form-urlencoded

Name Type Required Description
html String yes HTML code (including CSS)
filename String no File name, used during download

Example: Show PDF

POST /pdf/down Download PDF

Please refer to /pdf for parameters

POST /compact-pdf Generate a PDF file

Please refer to /pdf for parameters

To facilitate file operations, please point the/tmp/pdf directory of the Docker container to the folder on the host computer.
docker run -itd --name pdf-server -p 8000:80 -v /my/work/dir:/tmp/pdf ziyo-pdf-server
